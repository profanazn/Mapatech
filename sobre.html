Sobre o Projeto

O Mapa Tech é um projeto educacional desenvolvido para apresentar aos alunos do Ensino Médio Técnico as principais áreas da tecnologia e incentivar a exploração de carreiras na área de TI.

HTML

Responsável pela estrutura e organização das páginas do site.

CSS

Define cores, fontes, espaçamentos, layout e responsividade da aplicação.

JavaScript

Será utilizado para tornar o site interativo, adicionando navegação, quiz e outras funcionalidades nas próximas versões.


Este projeto também será utilizado nas aulas para demonstrar a evolução de um software utilizando Git e GitHub, registrando cada alteração por meio de commits.
